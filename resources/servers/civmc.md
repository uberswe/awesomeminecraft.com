# CivMC

**Website:** [https://civmc.net](https://civmc.net)

**Category:** Servers > Geopolitical Servers

---

## Overview

Completely free vanilla server continuing a decade-long tradition of "civ" servers focused on political simulation. Unique plugins create emergent gameplay without any monetization.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Java |
| **Audience** | All |
| **Price** | Free |
| **Server IP** | play.civmc.net |

## Features

- Citadel plugin for block reinforcement
- ExilePearl for player imprisonment
- FactoryMod for tech trees
- 50-150 concurrent players
- Community-funded with no store
- Entirely player-driven economy
