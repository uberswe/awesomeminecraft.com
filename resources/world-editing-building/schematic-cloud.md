# Schematic Cloud

**Website:** [https://schematic.cloud/](https://schematic.cloud/)

**Category:** World Editing & Building > Schematic Hosting

---

## Overview

Cloud-based schematic hosting service used by some Minecraft servers to automatically upload WorldEdit schematics. Provides storage and sharing for schematic files.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Web |
| **Audience** | Intermediate |
| **Price** | Free |

## Features

- Automatic schematic uploads from servers
- WorldEdit integration
- Shareable schematic links
- Used by various Minecraft server networks
