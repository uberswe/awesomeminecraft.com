# Schematic Lab

**Website:** [https://www.schematiclab.io/](https://www.schematiclab.io/)

**Category:** World Editing & Building > Schematic Tools

---

## Overview

Web-based tool for converting Minecraft schematics between different formats and versions. Upload schematic files and convert them to different output formats or target different Minecraft versions.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Web |
| **Audience** | All |
| **Price** | Free |

## Features

- Convert between schematic formats
- Target different Minecraft versions
- Browser-based, no installation required
- Support for multiple schematic types
