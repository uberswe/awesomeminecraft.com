## Overview

Comprehensive database of vanilla and modded Minecraft blocks and items. A useful reference for finding block IDs, properties, and information across different mods.

| Property | Value |
|----------|-------|

## Features

- Vanilla Minecraft blocks and items database
- Modded content support
- Block properties and IDs
- Search and browse functionality
