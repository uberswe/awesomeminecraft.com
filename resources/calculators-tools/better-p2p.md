**Website:** [https://www.curseforge.com/minecraft/mc-mods/betterp2p](https://www.curseforge.com/minecraft/mc-mods/betterp2p)

## Overview

P2P tunnel management mod for Applied Energistics 2. The "Better Memory Card" shows all P2P devices in your network from any connected device with visual highlighting for frequencies and errors.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Java |
| **Audience** | All |
| **Price** | Free |

## Features

- Visual highlighting for matching frequencies (aqua)
- Red errors for wrong setup, yellow warnings for no channel
- Input P2P always displays on top for clarity
- Versions: 1.20.1 Fabric/Forge, 1.21
- GTNH maintains 1.7.10 version
