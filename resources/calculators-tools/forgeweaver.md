# ForgeWeaver

**Website:** [https://www.forgeweaver.io/minecraft](https://www.forgeweaver.io/minecraft)

**Category:** Calculators & Tools > Production Planning

---

## Overview

Node-based diagram tool for building recipe diagrams for any game including modded Minecraft. Replaces spreadsheets for complex production chains with automatic resource calculation.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Web |
| **Audience** | All |
| **Price** | Free (Beta) |

## Features

- Build node-based recipe diagrams
- Automatically calculates resource totals
- Tracks cross-recipe dependencies
- Works with ANY modpack (manual recipe entry)
- Organize by world/server/modpack
- Share boards with teams
- No database needed
