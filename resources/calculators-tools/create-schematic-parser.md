# Create Schematic Parser

**Website:** [https://mimexe.github.io/CreateSchematicParser/](https://mimexe.github.io/CreateSchematicParser/)

**Category:** Calculators & Tools > Create Mod Tools

---

## Overview

Analyzes Minecraft Create mod schematic files (.nbt format) to discover which mods and Minecraft versions are required for the builds contained within them.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Web |
| **Audience** | Intermediate |
| **Price** | Free |

## Features

- Upload and analyze Create mod schematics
- Identifies required mods for builds
- Detects Minecraft version requirements
- Browser-based with local storage persistence
- Beta version with ongoing development
