# CalcMod

**Website:** [https://modrinth.com/plugin/calcmod](https://modrinth.com/plugin/calcmod)

**Category:** Calculators & Tools > Universal Tools

---

## Overview

Chat-based calculation mod that provides material breakdowns directly in chat. Use commands for coordinate conversions, comparator signal calculations, farm output rates, and custom functions.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Java |
| **Audience** | All |
| **Price** | Free |

## Features

- `/calc craft <item> <amount>` for material breakdowns
- Coordinate conversions between dimensions
- Comparator signal calculations
- Farm output rate calculations
- Custom functions support
- Works client-side without server installation
- Supports Fabric, Forge, Quilt, Paper (MC 1.12+)
