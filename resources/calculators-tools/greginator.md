## Overview

Comprehensive web app containing seven calculation tools for GregTech including overclock calculator, turbine calculator, tool browser, and more.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Web |
| **Audience** | All |
| **Price** | Free |

## Features

- Overclock Calculator with GT++ multiblock and batch mode support
- Turbine Calculator
- GT Tool Browser (sortable by durability/speed/damage)
- Cable Browser with amp/loss data
- Ore Vein Finder by coordinates
- Food Browser (1,090 items)
- Cell Calculator and Recipe Conflict Checker
- Supports Infitech 2, Infitech 3, and GTNH
- Dark/light themes
