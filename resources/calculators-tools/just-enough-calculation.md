# Just Enough Calculation (JEC)

**Website:** [https://www.curseforge.com/minecraft/mc-mods/just-enough-calculation](https://www.curseforge.com/minecraft/mc-mods/just-enough-calculation)

**Category:** Calculators & Tools > Universal Tools

---

## Overview

Essential in-game mod that recursively calculates total resources needed for any recipe. With over 20.4 million downloads, this is a must-have for any modded Minecraft player.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Java |
| **Audience** | All |
| **Price** | Free |

## Features

- Click the "+" button in JEI/REI to import any item
- Breaks down every sub-component to base materials
- Scans inventory for missing ingredients
- Supports fluids and recipe grouping
- Works with Forge, Fabric, Quilt, and NeoForge
- Versions: MC 1.7.10 through 1.20.1
