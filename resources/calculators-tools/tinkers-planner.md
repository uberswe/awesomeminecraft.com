# Tinker's Planner

**Website:** [https://www.curseforge.com/minecraft/mc-mods/tinkers-planner](https://www.curseforge.com/minecraft/mc-mods/tinkers-planner)

**Category:** Calculators & Tools > Tinkers' Construct

---

## Overview

Must-have in-game mod that integrates directly into the Tinker Station GUI to preview tool stats before crafting. With 3.9 million downloads, it's essential for Tinkers' Construct players.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Java |
| **Audience** | All |
| **Price** | Free |

## Features

- Test modifier combinations before crafting
- Bookmark favorite designs
- Randomizer for experimentation
- Auto-places matching parts from inventory
- Can import existing tools for comparison
- Tinker's Planner Reborn available for Tinkers' Construct 3
