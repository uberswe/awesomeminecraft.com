# GTNH Calculator

**Website:** [https://shadowtheage.github.io/gtnh/](https://shadowtheage.github.io/gtnh/)

**Category:** Calculators & Tools > GregTech Tools

## Overview

The best web-based production planner for GregTech New Horizons. Downloads complete recipe data to enable full production chain calculations with a Minecraft-style UI showing actual item icons.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Web |
| **Audience** | All |
| **Price** | Free |

## Features

- Downloads ~30MB of complete recipe data from GTNH
- Minecraft-style UI with actual item icons
- Designs save to browser local storage or shareable URLs
- Use negative amounts to specify output goals
- Links to NEI, wiki, and official spreadsheets
- Actively maintained for GTNH 2.8.0
