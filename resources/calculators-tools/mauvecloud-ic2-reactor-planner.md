## Overview

The definitive IC2 reactor tool supporting EU and Fluid reactors with simple, pulsed, and automated cycle modes. The standard planning tool for IndustrialCraft 2 nuclear reactors.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Java App |
| **Audience** | All |
| **Price** | Free |

## Features

- Drag-drop component placement
- Heat distribution modeling
- Component damage tracking and vent cooling calculations
- Export designs as Base64 codes for sharing
- Import from legacy Talonius v3 codes
- Export CSV data for graphing
- Includes GregTech 5 Unofficial component support
