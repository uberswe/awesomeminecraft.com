## Overview

Python CLI tool that generates PNG/PDF flowcharts for complex GregTech processing lines like platline, petrochemistry, and rare earth processing.

## Details

| Property | Value |
|----------|-------|
| **Platform** | CLI |
| **Audience** | Advanced |
| **Price** | Free |

## Features

- Automatically calculates machine quantities
- Balances production rates and handles overclock math
- Recognizes GT++ multiblock names for automatic tier adjustments
- PDF output is fully searchable
