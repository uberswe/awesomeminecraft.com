# Just Enough Immersive Multiblocks

**Website:** [https://modrinth.com/mod/jei-multiblocks](https://modrinth.com/mod/jei-multiblocks)

**Category:** Calculators & Tools > Immersive Engineering

---

## Overview

Adds JEI/EMI support for all Immersive Engineering multiblocks. Search for multiblocks directly in JEI, view structures and ingredients, and navigate from catalyst ingredients to recipes.

## Details

| Property | Value |
|----------|-------|
| **Platform** | Java |
| **Audience** | All |
| **Price** | Free |

## Features

- Search multiblocks in JEI
- View structures and ingredients
- Navigate from catalyst to recipes
- Works with IE addons like Immersive Petroleum
- Over 13.1 million downloads
