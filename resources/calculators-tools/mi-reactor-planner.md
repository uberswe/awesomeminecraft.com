## Overview

Clean web-based nuclear reactor planner for Modern Industrialization, optimizing for deuterium production, tritium production, plutonium breeding, or raw power (EU/t).

## Details

| Property | Value |
|----------|-------|
| **Platform** | Web |
| **Audience** | All |
| **Price** | Free |

## Features

- Supports all reactor sizes (Small, Medium, Large, Extreme)
- URL-based state sharing for design links
- Includes community example builds
- Optimization for multiple outputs
- MIT licensed
